# 💰 Expense Tracker

A modern **AI-powered Expense Tracker web app** that helps you manage money smarter, track spending habits, and achieve savings goals — all with a beautiful, futuristic dashboard UI.

---

## ✨ Features

💸 **Add Transactions**

* Income & Expense tracking
* Category-based entries (Food, Bills, Travel, etc.)
* Real-time updates

📊 **Smart Analytics**

* Income vs Expense doughnut chart
* Category-wise spending pie chart
* Visual financial insights

🎯 **Savings Goal System**

* Set custom financial goals
* Live progress tracking
* Auto goal completion detection
* Confetti celebration 🎉

📚 **Goal History**

* Stores all completed goals
* Date-wise tracking of achievements

🏆 **Achievements System**

* 10 transactions badge
* Power user badge (50+ transactions)
* Finance legend badge (100+ transactions)
* Goal master badge

🤖 **AI Insights**

* Finds your top spending category
* Gives smart saving suggestions
* Total expense breakdown

🔍 **Smart Filters**

* Search transactions
* Category filter system

💾 **Local Storage Support**

* All data saved in browser
* No backend required
